home module¶

home.show_home()¶

Function: show_home Handles the homepage route for Rev’s American Grill.

  • Retrieves user info, including reward points if logged in.
  • Fetches ‘Aggie Favorites’ and seasonal menu items based on the current weather.
  • Renders the homepage with user data and menu recommendations.
home.signup()¶

Function: signup Handles user signup via a POST request.

  • Retrieves form data (name, phone, email).
  • Connects to the database and checks if the phone or email already exists.
  • Returns an error message if the phone or email already exists.
  • Inserts a new customer into the database with initial reward points if validation passes.
  • Returns a JSON response indicating success or failure.
