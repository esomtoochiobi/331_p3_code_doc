salesPage module¶

salesPage.get_data(start_date, end_date, selected_category)¶

Creates SQL commands given the start_date, end_date, and selected_category parameters, and returns dictionaries that can be used to construct the graphs.

salesPage.show_salesPage()¶

Gathers sales data based on current day, and passes it into main sales page HTML file.

salesPage.update_graph()¶

Used to update the graph on the sales page.
