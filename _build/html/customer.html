customer module¶

customer.get_order()¶

Takes in a list of dictionaries, cart, which contains information about the items ordered by the customer, including menu id, name, quantity, and price. It then calls the function to update the databases. A message is returned indicating if the order was placed successfully or not

customer.get_points()¶
This function gets the updated Rev’s reward points from the Javascript function sendPoints(), and updates the customer’s points
attribute in the customer database
customer.show_customer()¶

This function sends data to the customer.html file with the following information: a list of menu items, information about the user including employee status and points, a list of categories, customer’s login status, and the current Rev’s reward points the customer currently has.

customer.update_inventory_quantities(cart_data)¶

This function updates the orders, menu_orders, and inventory databases. It checks if the person placing the order is an employee, and only completes the order if not. It checks if any of the inventory items would go below 0 for an item, and does not order that item if that is the case. It returns a message indicating if the order was placed correctly, and if not, which items were not ordered.
