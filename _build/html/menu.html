menu module¶

menu.show_menu()¶

Accesses the menu in the database and passes the data into the menu page. The first image in the database has been chosen to be shown - this can be changed if needed relatively easily.

menu.show_menuBoard()¶

VERY similar to show_menu, except this is meant to be pulled up in a seperate window popup. Does not contain the navbar or footer elements.
