inventoryManagement module¶

inventoryManagement.add_item_route()¶

Adds a new inventory item to the database.

Returns:Redirects to the inventory management page.
inventoryManagement.archive_item_route(ingredient_id)¶

Archives an inventory item.

Parameters:ingredient_id (int) – The ID of the inventory item to archive.
Returns:Redirects to the inventory management page.
inventoryManagement.edit_item_route(item_id)¶

Edits an existing inventory item.

Parameters:item_id (int) – The ID of the inventory item to edit.
Returns:JSON object containing details of the inventory item for editing or JSON response indicating success or failure of the update operation.
inventoryManagement.show_inventoryManagement()¶

Renders the inventory management page.

Returns:HTML template for the inventory management page populated with inventory data.
inventoryManagement.unarchive_item_route(ingredient_id)¶

Unarchives an inventory item.

Parameters:ingredient_id (int) – The ID of the inventory item to unarchive.
Returns:Redirects to the inventory management page.
