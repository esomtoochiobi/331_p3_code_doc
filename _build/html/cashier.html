cashier module¶

cashier.cancel_order()¶

This function is the method for cancelling an order in the kitchen view. This toggles the status of the order.

cashier.edit_order()¶

This is function is the post method of editing an order. When the order is incomplete, the items in the order can still be edited and are done through this method. The items in the order are loaded back into the front end where they can be edited and then correctly sent to the menu order table.

cashier.finish_order()¶

This is function is the post method of finishing an order. This toggles the status of the order so that when in the kitchen view, the cashier can complete an order and update the database.

cashier.place_order()¶

This is function is the post method of placing an order. When this function is called it routes the incoming order data including the items, customer, and exmployee back into the database so that the orders, menu_order, inventory, and customer data bases can be updated with the correct information.

cashier.show_cashier()¶

This function is the entry point of the cashier page. It accesses the DB to get the entire menu, list of customers, and the employee information. The data is used to load the menu into the cashier side, so that when items are added and or removed they can be accessed. Additionally, it can check whether customers exist to either add them, or create a new customer with the given information. Essentially, this function is the control point for this page.
